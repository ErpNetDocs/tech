# Download files from an invoice

Use this procedure to download files attached to an invoice in Client Center.

Invoice files are available only when the invoice contains attachments that are shared for both internal and external users in ERP.net. If no such files are available, the **Files** section does not appear in the invoice.

[screenshot: client-center/billing/operations/download-files-from-an-invoice/cc-billing-operations-download-files-from-an-invoice-01-invoice-files-section.png]

## Before you begin

Make sure that:

- the user has access to the **Billing** section;
- the correct customer is selected when the user has access to multiple customers;
- the required invoice is visible in Client Center;
- the invoice contains attached files that are shared for external access.

A file becomes visible in Client Center only when its **Access Permissions** field allows access to both internal and external users.

For more information, see:

- [Billing overview](../index.md)
- [Invoices page](../concepts/invoices-page.md)
- [File visibility and downloads](../../concepts/file-visibility-and-downloads.md)

## Download a file from an invoice

### 1. Open the Billing section

Open **Billing** and locate the invoice whose files you want to download.

If needed, use the available grid functions to find the required invoice more quickly.

For more information about the shared list behavior, see [Shared grid behavior](../../concepts/shared-grid-behavior.md).

### 2. Open the invoice document

Select the invoice to open its details.

Review the document and locate the **Files** section.

If the invoice contains externally shared attachments, the section shows the available files together with basic information such as file name, file extension, and size.

[screenshot: client-center/billing/operations/download-files-from-an-invoice/cc-billing-operations-download-files-from-an-invoice-02-open-invoice-document.png]

### 3. Download the file

In the **Files** section, select the file that you want to download.

Client Center downloads the selected file directly from the invoice document.

### 4. Repeat if needed

If more than one file is available, repeat the action for each required file.

[screenshot: client-center/billing/operations/download-files-from-an-invoice/cc-billing-operations-download-files-from-an-invoice-03-download-file.png]

## If the Files section is not shown

If the **Files** section does not appear in the invoice document, this usually means that the invoice does not contain any attachments that are shared for external access.

This can happen when:

- the invoice has no attached files;
- the invoice has attached files, but their **Access Permissions** do not allow access for external users.

For more information, see:

- [File visibility and downloads](../../concepts/file-visibility-and-downloads.md)
- [Files are not visible or downloadable](../../troubleshooting/files-are-not-visible-or-downloadable.md)

## Result

After the file is selected, it is downloaded from the invoice document to the user's device.
