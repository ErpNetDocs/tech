# Operations

Use the pages in this section for the main tasks that users perform in the **Billing** area of Client Center.

At this level, the Billing operations focus on working with files attached to invoice documents.

The exact availability of these actions depends on the user's external role and on whether the relevant invoice files are shared for external access.

## In this section

- [Download files from an invoice](download-files-from-an-invoice.md)  
  Explains how to download files attached to an invoice document when those files are shared for external access.

## Before you begin

Before performing the tasks in this section, make sure that:

- the user has the required role to access **Billing**;
- the correct customer is selected when the user has access to multiple customers;
- the required invoice is visible in Client Center;
- the relevant file is shared for both internal and external users.
