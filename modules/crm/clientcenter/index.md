# Client Center

Client Center is a web-based portal that gives external users controlled access to customer-specific information and actions in ERP.net.

It is designed for scenarios where customers, partners, or other external contacts need to work with selected business data without requiring access to the full ERP system. Depending on the configured access rights and enabled features, users can review orders, invoices, due payments, payment history, customer profile information, and related files.

## Why use Client Center

Client Center helps organizations expose selected ERP.net functionality to external users in a controlled and focused way. Instead of granting broad system access, it provides a dedicated interface for specific customer-facing activities.

Typical use cases include:

- reviewing submitted orders;
- creating new orders, when ordering is enabled;
- checking invoices and payment obligations;
- downloading documents and related files;
- managing external user access;
- reviewing customer profile information.

## When to use it

Use Client Center when external users need self-service access to business information that is maintained in ERP.net.

Examples include:

- customers tracking their orders and invoices;
- customers downloading invoice attachments or other shared files;
- organizations allowing external users to submit orders online;
- administrators managing which external users can access which customer data.

## Main areas

Client Center includes several functional areas.

### Orders

The **Orders** area provides access to submitted orders and, when enabled, allows users to create new ones.

For more information, see [Orders](orders/index.md).

### Billing

The **Billing** area gives users access to invoices, due payments, and payment history.

For more information, see [Billing](billing/index.md).

### User Management

The **User Management** area is used to manage external users and their access to customer data.

For more information, see [User Management](user-management/index.md).

### Customer Profile

The **Customer Profile** area displays customer-specific information and, depending on the configuration, may also provide access to shared files.

For more information, see [Customer Profile](customer-profile/index.md).
