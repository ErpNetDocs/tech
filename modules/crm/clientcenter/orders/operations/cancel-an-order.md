# Cancel an order

Use this procedure to cancel an order in Client Center before it is released.

Order cancellation is available only for orders that have already been submitted but have not yet been released. Once an order is released, it can no longer be canceled from Client Center.

[screenshot: client-center/orders/operations/cancel-an-order/cc-orders-operations-cancel-an-order-01-order-details.png]

## Before you begin

Make sure that:

- the user has access to the **Orders** section;
- the correct customer is selected when the user has access to multiple customers;
- the order has already been submitted;
- the order has not yet been released.

If the order is already released, the cancel action is no longer available.

For more information, see:

- [Orders overview](../index.md)
- [Orders page](../concepts/orders-page.md)
- [Multi-customer login](../../concepts/multi-customer-login.md)

## Cancel the order

### 1. Open the Orders section

Open **Orders** and locate the order that you want to cancel.

If needed, use the available grid functions to find the required document more quickly.

For more information about the shared list behavior, see [Shared grid behavior](../../concepts/shared-grid-behavior.md).

### 2. Open the order document

Select the order to open its details.

Review the order and confirm that it is the correct document to cancel.

[screenshot: client-center/orders/operations/cancel-an-order/cc-orders-operations-cancel-an-order-02-open-order.png]

### 3. Cancel the order

If the order is still eligible for cancellation, use the available cancel action in the order document.

Client Center allows cancellation only before the order is released.

### 4. Confirm the cancellation

Confirm the cancellation when prompted.

After confirmation, the order is canceled in ERP.net.

[screenshot: client-center/orders/operations/cancel-an-order/cc-orders-operations-cancel-an-order-03-confirm-cancel.png]

## Result

After the cancellation is confirmed:

- the order is marked as canceled;
- the canceled order remains visible in the Orders section;
- the order can no longer continue through the normal release flow.
