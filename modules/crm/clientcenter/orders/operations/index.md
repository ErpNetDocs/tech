# Operations

Use the pages in this section for the main tasks that users perform in the **Orders** area of Client Center.

These tasks include creating a new order, canceling a submitted order before release, and downloading files attached to an order document.

The exact availability of these actions depends on the user's external role and the Client Center configuration for the site.

## In this section

- [Create an order](create-an-order.md)  
  Explains how to create a new order in Client Center when the **New Order** function is enabled.

- [Cancel an order](cancel-an-order.md)  
  Explains when and how a submitted order can be canceled before it is released.

- [Download files from an order](download-files-from-an-order.md)  
  Explains how to download files attached to an order document when those files are shared for external access.

## Before you begin

Before performing the tasks in this section, make sure that:

- the user has the required role to access **Orders**;
- the Client Center configuration allows the corresponding action;
- the current customer context is correct when the user has access to multiple customers.

For example, **New Order** is available only when ordering is enabled in the Client Center settings and a valid order document type is configured.
