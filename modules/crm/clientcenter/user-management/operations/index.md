# Operations

Use the pages in this section for the main tasks that authorized users perform in the **User Management** area of Client Center.

These tasks focus on adding external users for the current customer and maintaining their access settings.

The exact availability of these actions depends on the user's external role. Only users with **L80 - Admin** or **L90 - Owner** can work with User Management.

## In this section

- [Add and manage external users](add-and-manage-external-users.md)  
  Explains how to add external users for the selected customer and how to maintain their access settings in Client Center.

## Before you begin

Before performing the tasks in this section, make sure that:

- the user has access to **User Management**;
- the correct customer is selected when the user has access to multiple customers;
- the external user account exists and can be assigned to the current customer.
