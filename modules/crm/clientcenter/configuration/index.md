# Configuration

This section describes the settings that control the visual appearance and functional behavior of Client Center.

> [!IMPORTANT]
> To apply a setting, restart the website using the Instance Manager.

## How settings work

Client Center settings are defined in the **Settings** field of the Client Center website definition in JSON format.

If a setting is not explicitly added, its rule is not enforced.

## Invalid configuration

The value of the **Settings** field must be valid JSON.

If the JSON is invalid or incomplete, Client Center cannot apply the configuration correctly.

For related problems, see [Settings changes are not applied](../troubleshooting/settings-changes-are-not-applied.md) and [Error codes](../troubleshooting/error-codes.md).
