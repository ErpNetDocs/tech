# Operations

Use the pages in this section for the main tasks that users perform in the **Customer Profile** area of Client Center.

At this level, the Customer Profile operations focus on working with customer-related files that are attached directly to the customer record and shared for external access.

The availability of these actions depends on the current customer context, the user's external role, and the sharing configuration of the file itself.

## In this section

- [Download files from Customer Profile](download-files-from-customer-profile.md)  
  Explains how to download files from the **Files** panel in Customer Profile when those files are shared for external access.

## Before you begin

Before performing the tasks in this section, make sure that:

- the user has access to **Customer Profile**;
- the correct customer is selected when the user has access to multiple customers;
- the required customer-related files are available in the profile;
- the files are shared for both internal and external users.
