# Download files from Customer Profile

Use this procedure to download files from the **Files** panel in Customer Profile.

Customer Profile files are available only when the customer record contains attachments that are shared for both internal and external users in ERP.net. If no such files are available, the Files panel does not show downloadable documents and the page shows the message **No documents available.**

[screenshot: client-center/customer-profile/operations/download-files-from-customer-profile/cc-customer-profile-operations-download-files-from-customer-profile-01-files-panel.png]

## Before you begin

Make sure that:

- the user has access to **Customer Profile**;
- the correct customer is selected when the user has access to multiple customers;
- the customer record contains attached files that are shared for external access;
- the required file is visible in the **Files** panel.

A file becomes visible in Client Center only when its **Access Permissions** field allows access to both internal and external users.

## Download a file from Customer Profile

### 1. Open Customer Profile

Open **Customer Profile** for the current customer.

Review the page and locate the **Files** panel.

[screenshot: client-center/customer-profile/operations/download-files-from-customer-profile/cc-customer-profile-operations-download-files-from-customer-profile-02-open-customer-profile.png]

### 2. Review the available files

If customer-related files are available for external access, the Files panel shows the available documents together with basic information such as file name, file extension, and size.

These files are attached directly to the customer record rather than to a specific order or invoice.

### 3. Download the file

In the **Files** panel, select the file that you want to download.

Client Center downloads the selected file directly from Customer Profile.

### 4. Repeat if needed

If more than one file is available, repeat the action for each required file.

[screenshot: client-center/customer-profile/operations/download-files-from-customer-profile/cc-customer-profile-operations-download-files-from-customer-profile-03-download-file.png]

## If no files are shown

If Customer Profile does not show downloadable files, this usually means that no customer attachments are currently shared for external access.

This can happen when:

- the customer record has no attached files;
- the customer record has attached files, but their **Access Permissions** do not allow access for external users.

In this case, the page shows the message **No documents available.**

For more information, see:

- [Files in Customer Profile](../concepts/files-in-customer-profile.md)
- [File visibility and downloads](../../concepts/file-visibility-and-downloads.md)
- [Files are not visible or downloadable](../../troubleshooting/files-are-not-visible-or-downloadable.md)

## Result

After the file is selected, it is downloaded from Customer Profile to the user's device.
