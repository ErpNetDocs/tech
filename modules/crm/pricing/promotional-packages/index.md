## Promotional Packages

**Promotional packages** are used to define predefined promotional offers that can be made available and applied during the sales process. They help organize promotional logic in a structured way, so that users can work with ready-to-use promotion bundles instead of configuring each promotional benefit separately.

A promotional package can include the rules and setup necessary for determining whether it is available for a specific sales document and for applying it when the required conditions are met. This makes promotional offers easier to maintain and more consistent across different sales scenarios.

For initial setup, see [Create basic promotional package](create-basic-promotional-package.md).

For configuration details, see [Configuration](configuration.md).

For the selection logic, see [Determine available promotional packages](concepts/determine-available-promotional-packages.md).

For the application process, see [Apply promotional packages](concepts/apply-promotional-packages.md).
