| --------------------- | ------------------------------------------------------------ |

| Parameter 1 Name      | String                                                         |
| Parameter 1 Type      | string                                    |
| Parameter 2 Name      | NumChars                                                           |
| Parameter 2 Type      | int                                                           |
| Parameter 3 Name      | -                                                            |
| Parameter 3 Type      | -                                                            |
| Return value          | Returns the last NumChars characters of String.                                                          |
