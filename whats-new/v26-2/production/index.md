### 1. Work Orders: New UI function Create completing output order

The existing Create completing output order function for Work Orders is now available as a UI function in both the Web and Desktop clients. 
The function can be executed for a single document from the Work Order form, or for multiple selected documents from the Work Orders navigator, and is also accessible through the API. 
It follows the same logic as the existing Create completing output order function and is intended to create completing output orders for distributing the remaining materials and undistributed cost.

This enables more efficient bulk processing of work orders from the user interface or through API integrations.
