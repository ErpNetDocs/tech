J38417 Delete old audit logs

Code J38417
Entity Systems.Core.AuditLogEntries
Job type name Delete old audit logs
Parameters /Monitoring/AuditLogEntriesRetentionMonths key
Description Deletes old audit log entries whose EventTimeUtc is older than 12 months.
The default value of 12 months can be adjusted according to the specific needs by configuring and setting a preferred value in the /Monitoring/AuditLogEntriesRetentionMonths key*.
