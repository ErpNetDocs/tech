    WARNING

    Typically used to show a non-blocking message. It may be useful when you want to show a reminder, instead of stopping the transaction with an error using a FAIL business rule. The WARNING syntax is simple - the message that will be shown is set as a first parameter for the WARNING action.

    Example:

    ActionNo: 1; Action Type: WARNING; Parameter1 Type: Constant; Parameter1 Value: 'warning message text'.
